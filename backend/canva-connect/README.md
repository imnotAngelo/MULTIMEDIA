# Canva Connect API Integration Backend

This service is scaffolded for integrating Canva with your system using the Canva Connect API.

## Next Steps

1. Clone the official Canva Connect API starter kit into this directory or copy the relevant files.
2. Register your app with Canva to obtain client ID and secret.
3. Configure OAuth and API endpoints as described in the Canva documentation.
4. Start the backend and connect it to your frontend for authentication and design management.
